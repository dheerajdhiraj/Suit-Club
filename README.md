# Suit Club — Premium E-commerce Website

A high-end, static e-commerce platform for bespoke suits, featuring a dynamic collection view, a shopping bag, and a secure checkout flow.

## ✨ Features
- **E-commerce Ready**: Add to Cart, Cart Management, and Checkout.
- **Dynamic Collections**: Categories and products are managed via a Mock API using `localStorage`.
- **Premium Design**: Modern, responsive UI with luxury-themed styling.
- **Admin Dashboard**: Effortlessly manage your collections and products (requires admin login).

## 🚀 Deployment Instructions

### Option 1: Vercel (Recommended)
1. Push your code to a **GitHub** repository.
2. Go to [Vercel](https://vercel.com/) and click **"Add New Project"**.
3. Import your GitHub repository.
4. Vercel will automatically detect the static site and deploy it.

### Option 2: GitHub Pages
1. Push your code to a **GitHub** repository.
2. Go to **Settings** > **Pages**.
3. Under **Build and deployment**, set the source to **"Deploy from a branch"**.
4. Select the `main` (or `master`) branch and click **Save**.
5. Your site will be live at `https://[your-username].github.io/[repo-name]/`.

---

## 🛠️ Local Development
Simply open `index.html` in any modern web browser. All data is persisted in your browser's `localStorage`.
